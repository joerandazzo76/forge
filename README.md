# Forge

Forge is a local-first AI dev orchestrator that turns product requests into disciplined software delivery workflows.

## v1 goals
- Create a project scaffold
- Persist local project state later
- Run milestone-based loops
- Validate work
- Create clean Git checkpoints

## Current status
This repository currently contains the first Rust CLI skeleton for Forge.

## Planned commands
- `forge new <project-name>`
- `forge status`
- `forge run`
- `forge next`
- `forge logs`
- `forge release`
